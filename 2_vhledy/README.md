# Vhledy

Výstupy sloužící prohlubování hlavních myšlenek, provádění experimentů a explorací, nebo i analýzy a metodiky, a samotným hypotézám, které jsou potřebné k prozkoumání.

## A - prohlubování znalostí

Slouží převážně k rozpracování a prohloubení myšlenek, pochopení mechanismů exokortexu a vlivu LLM na lidstvo, ze kterých se později skládají kognitivní výstupy. Jde současně i o ukázku co vše vzniká pro prohloubení myšlenky, aby se postavily jednotlivé části kognitivních výstupů. Jakmile vzniknou, přesouvají se prohloubené znalosti do složky archivu znalostí, aby se ukázalo na čem se aktuálně pracuje.
